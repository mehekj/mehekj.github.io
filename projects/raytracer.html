Raytracer

October 2021

I created this raytracer as a project for my Intro to Computer Graphics class in the 2021 fall semester. The project was programmed entirely in C++ in QtCreator.

Scenes are constructed of implicit primitives (cubes, spheres, cylinders, and cones) that can be transformed and placed (along with other scene components such as lights and the camera) using XML style scenefiles that are parsed and turned into scene graphs.

Features:
  • Full Phong illumination model including specular highlights
  • Recursively raytraced reflections
  • Cast shadows
  • Light attenuation
  • UV texture mapping
  • Camera transformations that can be adjusted by the user in the application GUI
